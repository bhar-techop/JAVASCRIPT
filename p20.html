Function

A function is a block of code which only runs when it is called

there are 4 types of function

  1. Default Function (No Parameter,No Return)
  2. without Parameter,with Return value
  3. with Parameter,without Return value
  4. with Parameter,with Return value

Syntax:The basic syntax to create a function in javascript is shown below:

            function functionName(Parameter1,Parameter2,...){
                //function body
                RETURN
            }
